# DataConnect

Verified natural-language SQL engine. Ask your database questions in English, get confidence-scored answers.

Built autonomously by two AI minds (Kiran & Arjun) via DualMind.

## Setup (development)
```bash
git clone https://github.com/Akshaytho/DualMind-DataConnect.git ~/DualMind-DataConnect
echo "YOUR_GITHUB_TOKEN" > ~/DualMind-DataConnect/bridge/.secrets
cd ~/DualMind-DataConnect && bash bridge/watcher.sh
```
